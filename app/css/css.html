CSS简介
  • CSS 指层叠样式表 (Cascading Style Sheets) 
  • 样式定义如何显示 HTML 元素 
  • 样式通常存储在样式表中 
  • 把样式添加到 HTML中，是为了解决内容与
   表现分离的问题 
  • 外部样式表可以极大提高工作效率 
  • 外部样式表通常存储在 CSS 文件中 
  • 多个样式定义可层叠为一
当同一个 HTML 元素被不止一个样式定义
时，优先级是咋样的？
  1.  浏览器缺省设置  
  2. 外部样式表  
  3. 内部样式表（位于 <head> 标签内部）  
  4. 内联样式（在 HTML 元素内部）
怎么写css
selector {
       property: value;
}
selector简介
P：标签选择器
       P标签选择器选择所有P标签元素
.student：class选择器
       选择所有标签里，class为student的元素
#password：id选择器
       选择所有id为password的元素

div .first #second：选择器可以组合使用
       选择div标签里的class为first元素当中
       id为second的元素
注意点：选择器的嵌套使用会降低性能
比如这样一种嵌套：div .first #second
