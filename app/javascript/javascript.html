}

js面向对象机制简述
每一个function是一个独立的对象，一个
函数可以作为另一个函数的参数
prototype：原型法设计模式，返回对象
类型原型的引用
new关键字： 以new的方式运行的话，函
数会被当成一个构造函数，从而得到一个
对象。也可以说，new是一个运算符，这
个运算符的执行结果是一个对象。
Object对象是其他所有JS对象的基类
